## 2. 编辑器（新建文章）

- [ ] NavBar 点 `+` → 进入编辑页
- [ ] 标题输入框可用
- [ ] Markdown 模式下打字 → 右侧实时预览
- [ ] 切换 Typst 模式 → 预览正常
- [ ] 分隔条拖拽（20%-80%）
- [ ] 下载源码按钮（FileDown 图标）→ 下载 `.md` / `.typ` 文件
- [ ] 下载 PDF 按钮（FileText 图标）→ 下载编译后文件

### 2.1 发布流程

- [ ] 点"发布"按钮 → 自评弹窗出现
- [ ] 输入 commit message（必填）→ 不写不能提交
- [ ] 五维自评星星可点击（O/R/C/P/I）
- [ ] 标签、领域、摘要可填
- [ ] "发布到沉淀池" → 成功后跳转文章页
- [ ] 文章状态显示 "沉淀中"

## 4. 文章页——评论（核心）

### 4.1 未登录
- [ ] 看到 "Sign in to submit a review"
- [ ] 如果有评审 → 能看到评审卡片（只读）
- [ ] 能看到 Thread 内容（只读）

### 4.2 登录——提交评审
- [ ] 非作者登录 → 看到评审表单（星星 + 文本框 + Submit）
- [ ] 五星可点击，金色星星显示正确
- [ ] 星星是**尖锐五角星**，不圆
- [ ] 输入评论文字
- [ ] Submit Review → 成功
- [ ] 提交后表单消失，评审卡片出现

### 4.3 登录——自己的评审
- [ ] 自己的评审**置顶**，accent 色左边框
- [ ] 显示 "(you)" 标签
- [ ] hover 分数（如 `O:4`）→ 展开为可编辑星星
- [ ] 鼠标移开 → 恢复数字显示
- [ ] 点击星星修改分数 → 反应即时（乐观更新）
- [ ] 修改失败时 → 显示红色错误提示

### 4.4 Thread 对话
- [ ] 自己的评审无 Thread → 显示 "Start a conversation..." 输入框
- [ ] 输入文字 + 回车 → 发送成功
- [ ] 发送失败 → "Failed to send" 红色提示
- [ ] 有 Thread 消息 → 显示 "Thread (N)" 按钮 + chevron
- [ ] 点 chevron → 展开/折叠
- [ ] 消息气泡 iMessage 风格（作者左、回复者右）
- [ ] 消息显示发送者名字（非 UUID）
- [ ] 作者可在所有评审下回复
- [ ] 评审人可在自己的评审下回复
- [ ] 旁观者看到 "Only the author and reviewer can participate"
- [ ] 旁观者无回复输入框

## 12. 边界情况

- [ ] 文章不存在 → "Article not found"
- [ ] 网络断开时提交评审 → 错误提示
- [ ] 超长标题 → 不溢出
- [ ] 空内容提交 → 后端拒绝
- [ ] 极长内容 → 编译不会挂
- [ ] 特殊字符（中文、数学符号）→ 显示正常
- [ ] 快速连续点击 → 不乱
