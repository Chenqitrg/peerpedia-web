# i18n 中英文切换

## 目标

在导航栏添加中/英文切换按钮，使用户可切换界面语言。五维评分标签（originality, rigor, completeness, pedagogy, impact）保持英文。

## 技术方案

安装 `vue-i18n`，创建 `zh-CN` 和 `en-US` 两个 locale 文件，在 `main.ts` 注册插件。语言选择存 `localStorage`，刷新保持。

## 语言切换按钮

放在 NavBar 搜索栏和书签按钮之间，样式与其他导航图标按钮一致：

```html
<button @click="toggleLocale" aria-label="Switch language">
  {{ currentLocale === 'zh-CN' ? 'EN' : '中' }}
</button>
```

## 文件结构

```
frontend/src/
├── locales/
│   ├── en-US.json    ← English (fallback)
│   └── zh-CN.json    ← 中文
├── main.ts           ← 注册 createI18n
```

## 依赖

```json
"vue-i18n": "^11.x"
```
