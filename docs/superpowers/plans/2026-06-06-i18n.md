# i18n 中英文切换 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为知诸网（PeerPedia）前端添加中英文切换功能，按钮在 NavBar，覆盖所有 UI 文案（按钮、提示、页面标题），保留五维评分英文。

**Architecture:** 安装 vue-i18n@11，创建 zh-CN/en-US locale JSON 文件，在 main.ts 注册插件。语言选择存 localStorage。所有组件中的硬编码英文用 `$t('key')` 替换。中文字体通过 CSS font-family 回退链自动匹配，不需 JS 干预。

**Tech Stack:** Vue 3 + vue-i18n@11 + Pinia + Vue Router

**Spec:** `docs/superpowers/specs/2026-06-06-i18n-design.md`

### Task 3: 注册 i18n 插件 + 中文字体

**Files:**
- Modify: `frontend/src/main.ts`
- Modify: `frontend/src/assets/main.css`

- [ ] **Step 1: 在 main.ts 中注册 vue-i18n**

```ts
import './assets/main.css'
import { createApp } from 'vue'
import { createPinia } from 'pinia'
import { createI18n } from 'vue-i18n'
import { router } from './router'
import App from './App.vue'
import enUS from './locales/en-US.json'
import zhCN from './locales/zh-CN.json'

// 从 localStorage 读取用户语言偏好
const savedLocale = localStorage.getItem('locale') || 'zh-CN'

const i18n = createI18n({
  legacy: false,        // 使用 Composition API 模式
  locale: savedLocale,
  fallbackLocale: 'en-US',
  messages: { 'en-US': enUS, 'zh-CN': zhCN },
})

const app = createApp(App)
app.use(createPinia())
app.use(router)
app.use(i18n)
app.mount('#app')
```

- [ ] **Step 2: 在 main.css 末尾添加中文字体引入和回退链**

```css
/* ── i18n fonts: Chinese + English bilingual ── */
@import url('https://fonts.googleapis.com/css2?family=Noto+Serif+SC:wght@400;500;600;700&display=swap');

/* 更新全局字体回退链，确保中英文各自用对应字体 */
body {
    font-family: 'Inter', 'PingFang SC', 'Microsoft YaHei', system-ui, sans-serif;
}

h1, h2, h3, h4, h5, h6, .font-heading {
    font-family: 'EB Garamond', 'Noto Serif SC', 'STSong', Georgia, serif;
}
```

### Task 4: NavBar 语言切换按钮 + 文案替换

**Files:**
- Modify: `frontend/src/components/NavBar.vue`

- [ ] **Step 1: 在 NavBar 添加语言切换逻辑和按钮**

在 `<script setup>` 中添加：

```ts
import { useI18n } from 'vue-i18n'

const { t, locale } = useI18n()

function toggleLocale() {
  locale.value = locale.value === 'zh-CN' ? 'en-US' : 'zh-CN'
  localStorage.setItem('locale', locale.value)
}
```

- [ ] **Step 2: 在搜索栏右侧添加切换按钮**

在搜索 form 结束标签后、书签按钮前添加：

```html
<!-- Language toggle -->
<button
  class="flex items-center justify-center w-8 h-8 rounded-lg
         text-xs font-semibold
         text-ink-muted hover:text-ink hover:bg-[#21262d]
         transition-colors duration-200"
  :aria-label="locale === 'zh-CN' ? 'Switch to English' : '切换到中文'"
  @click="toggleLocale"
>
  {{ locale === 'zh-CN' ? 'EN' : '中' }}
</button>
```
